# How to Create Your App Icon

## Option 1: Use the HTML Template (Quick & Easy)

1. **Open the template:**
   - Open `app-icon-template.html` in your web browser
   - You'll see a brown espresso-themed icon with coffee cup emoji

2. **Take a screenshot:**
   - Use a screenshot tool to capture EXACTLY the icon (1024x1024px)
   - On Mac: Use built-in screenshot tool and crop precisely
   - Make sure it's exactly 1024x1024 pixels

3. **Add to Xcode:**
   - Open Xcode project
   - Go to `Assets.xcassets` → `AppIcon`
   - Drag the 1024x1024 image into the "1024pt" slot

## Option 2: Use Online Icon Generators (Recommended)

### Free Tools:
1. **Canva** (canva.com)
   - Sign up free
   - Search for "App Icon" template
   - Customize with coffee/espresso theme
   - Download as 1024x1024 PNG

2. **App Icon Generator** (appicon.co)
   - Upload a 1024x1024 image
   - Automatically generates all required sizes
   - Download and drag into Xcode

3. **Icon Kitchen** (icon.kitchen)
   - Create simple icons with shapes and text
   - Download iOS icon set

### Recommended Design:
- **Background:** Brown gradient (#7A5449 to #3D2E23)
- **Icon:** Coffee cup, espresso machine, or coffee bean
- **Text:** "ET" or coffee cup emoji ☕
- **Style:** Minimalist, clean, professional

## Option 3: Hire a Designer (Professional)

- **Fiverr:** $5-50 for app icon design
- **Upwork:** Professional designers
- **99designs:** Design contests

## Adding Icon to Xcode

1. Open your project in Xcode
2. Click on `Assets.xcassets` in the left sidebar
3. Click on `AppIcon`
4. Drag your 1024x1024 PNG image into the "1024pt" slot
5. Xcode will automatically generate other sizes

## Icon Requirements for App Store

- **Size:** 1024 x 1024 pixels
- **Format:** PNG (no transparency)
- **Color Space:** sRGB or P3
- **No alpha channel:** Must be fully opaque
- **Rounded corners:** iOS adds these automatically, don't add them yourself
